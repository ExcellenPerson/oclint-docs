Homebrew
========

macOS users can install and update OCLint from our `homebrew tap <https://github.com/oclint/homebrew-formulae>`_.

Installing OCLint
-----------------

.. code-block:: bash

    $ brew tap oclint/formulae
    $ brew install oclint

Updating OCLint
---------------

.. code-block:: bash

    $ brew update
    $ brew upgrade oclint
